# PaizoWebsiteSaver
A Python script to save various OrgPlay pages from your PaizoAccount locally

**EDIT THE SOURCE TO SET YOUR OUTPUT LOCATION IN LINE 10**

The tool uses three freely available Python modules: requests, BeautifulSoup and lxml.

To install those requirements, run
```
pip install -r requirements.txt
```

To run the script:

```
python PaizoWebsiteSaver.py
```
